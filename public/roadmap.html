BreezeGo SaaS Roadmap

Mapa de Ruta de Producto & MVP

Gestión e ingeniería de fases, sprints secuenciales interactivos, tableros de prioridad y QA del despliegue.

1. Alcance del MVP

El MVP de BreezeGo está diseñado para lanzar una suite transaccional de forwarding Miami-SJO ágil y funcional en el menor tiempo. Incluye 10 componentes clave:

✓ Listo 🔑 Onboarding Wizard (signup)

Multi-step OTP focus grid y suite Miami generation.

✓ Listo 🔑 Login Portal (login)

Acceso JWT y reseteo por hCaptcha.

✓ Listo 🎛️ Command Center (dashboard)

Copy card template, counter badges y notificaciones.

✓ Listo 🚚 Tracker GPS (tracking)

Interactive map coordinate ticker y driver messaging.

✓ Listo 🧮 Cotizador Aéreo (calculator)

Volumetric pricing, Hacienda taxes y CRC converter.

✓ Listo ⚡ Pre-Alert Invoice (prealerts)

PDF invoice drag-drop uploader y localStorage logic sync.

✓ Listo 🙋‍♂️ Help center (help)

Floating FAB widget, grids categorizados y buscador live.

✓ Listo 📱 Capsule Navigation

Grid responsivo que despliega bottom capsule nav en móvil.

2. Fases de Desarrollo

Planificamos la escala estructural del SaaS en tres fases incrementales para asegurar estabilidad y control:

Fase 1

Client MVP Core

Lanzamiento del onboarding multipaso, cotizador arancelario y rastreador en vivo satelital para clientes.

Fase 2

Operations Consolidation

Lanzamiento del Customs Operator Desk (admin.html), consola de base de datos (architecture.html) y syncing de fletes.

Fase 3

Automation & Scale

Pasarela de validación automática SINPE/BAC, cálculo óptimo de rutas de choferes por IA, y mobile scanners.

4. Prioridades de Lanzamiento

Estructuramos las prioridades de ingeniería para enfocar los esfuerzos en componentes críticos antes de resolver capas de automatización:

CRITICAL Security & Auth Core

JWT protected routes, OTP SMS grids y PostgreSQL RLS policies para bloquear accesos indebidos.

HIGH Calculadora & Tracker GPS

Estimación CIF, IVA arancelario, y ticker de velocidad satelital en el mapa tracker.

MEDIUM Help Center & Invoices

Expandable categories index widgets y S3 invoice uploader progress simulator.

LOW Roadmap & Sprints

Visual planning dev dashboards y Notion scrollspy indicator boards.

5. Roadmap Técnico

El mapa de setup de staging e infraestructura técnica sigue una jerarquía de dependencias estricta:

Paso 1
Boilerplate Next.js & Mapeo CSS

Scaffolding, font preloading, e inicialización de variables de diseño `:root` en CSS.

Paso 2
Instalación shadcn/ui Component Library

Ejecutar npx shadcn-ui@latest init para inyectar primitivas UI atómicas.

Paso 3
Configuración Base Supabase & RLS

Sincronización de esquemas relacionales PostgreSQL y habilitación de RLS con filtros auth.uid().

Paso 4
Protected Routes Edge middleware

Despliegue de interceptores de sesión JWT al Vercel Edge para bloquear accesos directos.

Paso 5
Vercel CDN Edge Deployment

Automatización de compilación estática global en milisegundos.

6. Preparación para el Lanzamiento

Protocolo obligatorio de QA y auditorías de rendimiento previas a la liberación en producción:

✓
Responsive Viewport testing

Validación de bottom capsule navigation transformations en viewports celulares por debajo de 900px.

✓
Cross-Page database Syncing

Auditoría de consistencia de localStorage para que el panel operador admin modifique tracker milestones en vivo.

✓
Security Edge Auditing

Validación de Edge middlewares para bloquear accesos directos a sub-rutas sin cookies de sesión.

✓
Volumetric pricing validation

Garantía de aranceles fletes y cálculo volumétrico bloqueado bajo estándares IATA.

7. Sprints Futuros & Roadmap

La arquitectura modular y escalable permite liberar las siguientes características en sprints post-lanzamiento:

S1 BAC Checkouts & SINPE Gateways

Auto-validación de CRC SINPE depósitos móviles por lectura de SMS para liberar fletes.

S2 Rutas AI de Reparto

Optimización inteligente de camiones según congestión en Costa Rica para predecir ETAs.

S3 Native Mobile Scanner App

Aplicación móvil híbrida con escáner de barras para agilizar bodegas Miami.
