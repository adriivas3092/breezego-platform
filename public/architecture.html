Plano Base de Datos

Esquemas de tablas relacionales, llaves primarias/foráneas y tipos de columnas.

TABLA

profiles

Información extendida de usuario y casillero SJO.

id UUID [PK, FK -> auth.users.id]
first_name VARCHAR(100)
last_name VARCHAR(100)
id_card_type VARCHAR(20)
id_card_num VARCHAR(50) [UNIQUE]
phone_number VARCHAR(30)
suite_code VARCHAR(30) [UNIQUE]
delivery_address TEXT
delivery_district VARCHAR(100)
delivery_class VARCHAR(20) [gam/rural]
freight_preference VARCHAR(20) [aerial/maritime]
created_at TIMESTAMP WITH TIME ZONE
TABLA

packages

Cargos importados en Miami y tránsito a aduanas.

id UUID [PK]
tracking_number VARCHAR(100) [UNIQUE]
breeze_code VARCHAR(30) [UNIQUE]
user_id UUID [FK -> auth.users.id]
description TEXT
vendor VARCHAR(100)
weight_lbs DECIMAL(10,2)
volume_height DECIMAL(10,2)
volume_width DECIMAL(10,2)
volume_length DECIMAL(10,2)
fob_value_usd DECIMAL(10,2)
tax_category VARCHAR(50)
status VARCHAR(30)
created_at TIMESTAMP WITH TIME ZONE
updated_at TIMESTAMP WITH TIME ZONE
TABLA

tracking_events

Historial cronológico e hitos satelitales GPS.

id UUID [PK]
package_id UUID [FK -> packages.id]
status VARCHAR(30)
message TEXT
latitude DECIMAL(10,8)
longitude DECIMAL(11,8)
driver_name VARCHAR(100)
driver_phone VARCHAR(30)
created_at TIMESTAMP WITH TIME ZONE
TABLA

pre_alerts

Prealertas y facturas de compras registradas.

id UUID [PK]
user_id UUID [FK -> auth.users.id]
tracking_number VARCHAR(100)
vendor VARCHAR(100)
fob_value_usd DECIMAL(10,2)
invoice_url TEXT [Supabase Bucket URL]
created_at TIMESTAMP WITH TIME ZONE
TABLA

invoices

Fletes, impuestos arancelarios e importes CIF.

id UUID [PK]
package_id UUID [FK -> packages.id, UNIQUE]
flete_cost_usd DECIMAL(10,2)
taxes_cost_usd DECIMAL(10,2)
insurance_usd DECIMAL(10,2)
total_cost_usd DECIMAL(10,2)
total_cost_crc DECIMAL(12,2) [515 exchange rate]
is_paid BOOLEAN [DEFAULT false]
payment_method VARCHAR(30) [sinpe/card]
paid_at TIMESTAMP WITH TIME ZONE
TABLA

notifications

Registro interno de notificaciones y alertas de usuario.

id UUID [PK]
user_id UUID [FK -> auth.users.id]
title VARCHAR(150)
message TEXT
is_read BOOLEAN [DEFAULT false]
created_at TIMESTAMP WITH TIME ZONE
TABLA

support_tickets

Gestión interna de helpdesk y reclamos de aduana.

id UUID [PK]
user_id UUID [FK -> auth.users.id]
subject VARCHAR(200)
description TEXT
status VARCHAR(30) [open/closed/solved]
priority VARCHAR(20) [low/medium/high]
created_at TIMESTAMP WITH TIME ZONE

PERMISOS: Cliente

Acciones y recursos asignados al rol seleccionado en la base de datos.

PostgreSQL **Row Level Security (RLS)** restringe las lecturas y escrituras de filas directamente en la capa de datos según el JWT del usuario autenticado, evitando cualquier manipulación del cliente en consola.
